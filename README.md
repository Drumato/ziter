# ziter

A toy iterator implementatino for Zig.

## Features

- builtin iterator implementation
    - [x] `ArrayIterator`
    - [x] `SliceIterator`
    - [ ] `MapIterator`
    - [ ] `FileIterator`
- funtions
    - [x] all
    - [x] any
    - [ ] chain
    - [ ] cycle
    - [x] enumerate( `enum_next()` )
    - [x] filter
    - [x] map
    - [ ] find
    - [ ] take_while
    - [x] foldl/foldr
    - [ ] for_each
